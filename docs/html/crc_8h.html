Подсчёт контрольной суммы CRC-16.

Данная функция выполняет подсчёт CRC-16 (ARC) табличным методом.
Полином : 0x8005
Начальное значение : 0xFFFF
CRC("123456789") : 0x4B37

Аргументы
[in]srcБлок данных, по которому требуется произвести расчёт CRC
[in]lenДлина src в байтах
[out]resultРезультат вычисления
[in]oldcrcЕсли требуется произвести расчёт CRC на основе ранее вычисленного значения, передайте сюда адрес этого значения ; иначе NULL
Возвращаемые значения
ERR_NO
ERR_PARAM_INVALID
