err_type arch_getFileList ( FILE *  arch,
File_Header *  result,
uint64_t  maxCount,
uint64_t *  resultCount 
)

Функция получает список файлов из архива

Функция записывает в массив result список файлов ; чтобы получить весь список файлов, эту функцию следует вызывать до тех пор, пока она не вернёт значение ERR_EOF

Аргументы
[in]archОткрытый на чтение архив
[out]resultМассив, в который будет записан список файлов
[in]maxCountУказывает, информацию о скольких максимум файлах следует узнавать ; фактически, это значение не должно быть больше длины массива result
[out]resultCountУказывает, информация о скольких файлах была считана
Возвращаемые значения
ERR_NO
ERR_PARAM_INVALID
ERR_READ_FAILED
См. также
